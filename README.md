# Steam Checker
This is a Nametag mod for the game Gorilla Tag that shows a nametag above all players with the following info:

* Platform - if the player is on steam, standalone quest, or quest link.

* Fps - the fps or "hertz" of the player.
  
* Name - the players name.
  
* Color - the nametag is the color of the player and if they get tagged it becomes orange.

I might add icons for steam, quest, and quest link in the future but for now it just says it
Example: [Steam] [72] Gorilla1234
